# Lab 3 - Analysis
On this lab we did an exploratory data analysis (EDA) on a BMW sales dataset using Python. The goal is to analyze sales performance, identify patterns and visualize them.

## Tools & Libraries Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Analysis Performed
1️. Data Preprocessing
- Converted date columns to proper datetime format
- Created a Month column for monthly grouping
- Checked for missing values and data consistency

2️. Revenue Analysis
- Revenue grouped by Country
- Revenue grouped by Product
- Revenue grouped by Sales Person
- Displayed top-performing categories

3️. Correlation Analysis
- Generated a correlation matrix heatmap to examine relationships between:
- Boxes Shipped
- Amount

4️. Monthly Revenue Trend
- Aggregated revenue by month
- Plotted a time-series graph to visualize revenue trends over time
